FUNCTIONS

These are some of the functions running this web application

get_weather_data(city) - This function makes an API request to OpenWeatherMap to retrieve the current weather data for the given city. It then stores the weather data in the database table.

display_weather_data(city) - This function retrieves the weather data for the given city from the database table.

plot() - This function is called when the user submits the plot form. It retrieves the temperature data for the selected cities from the database table and creates a plot based on the selected plot option using Bokeh.

FEATURES

The application uses an API to retrieve current weather data for each city.

The application stores the weather data for each city in a SQLite database.

The application provides an option to plot the temperature data against the selected cities using various plot options such as scatter plot, bar chart, line plot, and histogram.

The application uses Bokeh to create interactive plots.
